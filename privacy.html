Privacy Policy

Last updated: March 30, 2026

This Privacy Policy describes how Aircraft Mechanic Tycoon handles information when you use the game.

Information We Collect

  • Gameplay data stored in your browser (for example money, upgrades, level, and settings).
  • Leaderboard data that you submit (for example character name, score, level, and total repairs).
  • Basic technical data provided by the platform or browser (for example device type, locale, and anonymous identifiers).

How Data Is Used

  • Save and restore your local game progress using browser storage.
  • Display and sync leaderboard rankings across devices when cloud sync is enabled.
  • Improve game performance and reliability.
  • Show advertisements through third-party game platforms and ad providers.

Storage And Retention

  • Local progress is stored in your browser via localStorage.
  • Leaderboard entries may be stored in a cloud database for ranking display.
  • You can clear local browser storage at any time from your browser settings.

Third-Party Services

The game may be distributed via third-party platforms (for example CrazyGames, GameDistribution, GamePix, Poki) and may use external services such as analytics, cloud databases, and ad providers. These services may process data under their own privacy policies.

Children

The game is intended for a general audience and does not knowingly collect sensitive personal information from children.

Your Choices

  • Use an alias/character name instead of your real name.
  • Do not submit leaderboard scores if you do not want cloud ranking data stored.
  • Clear local browser data to remove local game progress.
